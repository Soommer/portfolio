SklepAPI

Opis

Backend aplikacji restauracyjnej (kebab) umożliwiającej zamawianie online: konfigurację kebaba (mięso, sosy, dodatki), wybór typu dostawy i płatność. Panel administracyjny pozwala zarządzać menu, a pracownicy otrzymują zamówienia w czasie rzeczywistym dzięki SignalR.

Technologie:
  • C# 12, .NET 8 (ASP.NET Core Web API)
  • REST, JSON, obsługa plików, DI (.NET)
  • Dokumentacja: Swagger / OpenAPI (Swashbuckle)
  • Logowanie: Serilog
  • Konteneryzacja: Docker
  • Kontrola wersji: Git
  • Baza danych: Azure SQL Database
  • Przechowywanie plików: Azure Blob Storage
  • Płatności: Stripe
  • Real-time: SignalR
  • Mapy / odległości: Azure Maps
Hostowanie:

Azure Container Apps (backend) + Azure Static Web Apps (frontend).

Frontend:

Angular / TypeScript (autoryzacja JWT, integracja ze Stripe, SignalR).
