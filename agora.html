AgoraAPI

Opis

Backend forum internetowego umożliwiający rejestrację użytkowników, logowanie, zarządzanie profilem, tworzenie wątków i postów oraz ich moderację. Projekt zawiera pełną obsługę JWT, zarządzanie obrazami w Azure Blob Storage, dokumentację API w Swagger oraz testy jednostkowe i integracyjne.

Technologie:
  • C# 12, .NET 8 (ASP.NET Core Web API)
  • REST, JSON, obsługa plików
  • Dependency Injection (.NET Core DI)
  • Dokumentacja: Swagger / OpenAPI (Swashbuckle)
  • Baza danych: Azure SQL Database
  • Przechowywanie obrazów: Azure Blob Storage
  • Testy: xUnit, Moq
  • Autoryzacja: Microsoft.AspNetCore.Identity + JWT
  • Logowanie: Serilog
  • Konteneryzacja: Docker
  • Kontrola wersji: Git
Hostowanie:

Azure Container Apps (backend), Azure Static Web Apps (frontend).

Frontend:

React / TypeScript (obsługa JWT, integracja z API).
