AgoraAPI

Opis

Projekt Agora to prosty backend do forum internetowego, który umożliwia użytkownikom rejestrację, zażądzanie profilem, logowanie, tworzenie wątków i postów oraz zarządzanie nimi.

Technologie:
  • C# 12, .NET 8.0 (ASP.NET Core WebAPI)
  • REST, JSON, obsługa plików
  • Dependency Injection: .NET Core DI
  • Dokumentacja: Swagger / OpenAPI (Swashbuckle)
  • Baza danych: Azure SQL Database
  • Zdjęcia: Azure Blob Storage
  • Testowanie: xUnit, Moq, Microsoft.AspNetCore.Identity
  • Logowanie: Serilog
  • Konteneryzacja: Docker
  • Kontrola wersji: Git
Hostowanie:

Hosting w platformie azure za pomocą azure Container app.

Frontend:

React/ Typescript
