KryptografiaAPI

Opis

Projekt demonstracyjny API do szyfrowania i deszyfrowania wiadomości z użyciem klasycznych i współczesnych algorytmów (AES, RSA, SHA-256, Vigenère, Cezar), oraz steganografii (ukrywanie wiadomości/obrazów w plikach PNG).

Technologie:
  • C# 12, .NET 8 (ASP.NET Core Web API)
  • Algorytmy: AES, RSA, SHA-256, Vigenère, Cezar
  • Steganografia: LSB w PNG (ImageSharp)
  • Dokumentacja: Swagger / OpenAPI (Swashbuckle)
  • Logowanie: Serilog
  • Konteneryzacja: Docker
  • Kontrola wersji: Git
  • Frontend: HTML / CSS / TypeScript
Hostowanie:

Azure Container Apps (kontener z backendem i frontendem).
